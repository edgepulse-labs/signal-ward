# PCFA Icon 生成 Prompt

可將這份 prompt 用於 AI 圖像生成工具，替 PCFA 產生專案 icon。

## 主要 Prompt

Create a clean, modern browser-extension icon for "Personal Cognitive Firewall Assistant" (PCFA). The icon should communicate local-first cognitive protection, social-feed signal analysis, and calm attention defense. Use a simple shield outline combined with subtle signal-wave or neural-node motifs. Include a small checkmark or protected-focus cue, but avoid aggressive security imagery. The style should be flat vector-like, crisp at small sizes, high contrast, friendly, and trustworthy. Use a balanced palette with deep teal, fresh green, warm amber, and neutral off-white. No text, no letters, no screenshots, no people, no flags, no political symbols. Centered composition, transparent or solid light background, suitable for Chrome extension icon sizes 16x16, 32x32, 48x48, 128x128, and 512x512.

## Negative Prompt

Avoid: text, PCFA letters, tiny unreadable details, photorealistic faces, national flags, political party symbols, police or military imagery, harsh warning triangles, skulls, surveillance cameras, dark hacker aesthetics, complex gradients, cluttered dashboards, social media logos, X/Twitter logo, Threads logo.

## 視覺需求

- 適合作為小尺寸方形 icon。
- 在 16x16 仍有可辨識輪廓。
- 優先使用接近 vector 的清楚形狀與邊緣。
- 不使用社群平台品牌 logo。
- 不使用敏感政治或國家符號。
- 氣質應是友善保護，而不是懲罰式 moderation。

## 建議變體

1. 盾牌加三個訊號點，其中一個訊號點呈現被柔化或過濾。
2. 圓角盾牌中放入平靜的 focus mark 與小型 waveform。
3. Local-first lock-shield，背景有溫和 neural graph。
4. 極簡 teal 盾牌，搭配 amber signal pulse 與 green confirmation spark。
